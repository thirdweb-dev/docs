# Base Contracts

<div className="heroGrid">
  
  <div>
    <p>
      Base contracts provide implementations for a logical set of <a href='/extensions/features'>features</a> for you to utilize as the starting point for your smart contracts.
    </p>

Rather than implementing EIP standards manually, base contracts provide a logical foundation for you to extend and customize to your needs.

We provide several base contracts for [ERC721](/extensions/base-contracts/erc-721/erc721base), [ERC1155](/extensions/base-contracts/erc-1155/erc1155base), and
[ERC20](/extensions/base-contracts/erc-20/erc20base); each with different levels of functionality depending on your use case.

You can override and add additional functionality to the base logic, and include [features](/extensions/features) to supplement the base functionality.

## List of Base Contracts

Below is a list of all the base contracts available.

### ERC721

| Name                                                                          | Description                                                                                                                                                                                                                           |
| ----------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [ERC721Base](/extensions/base-contracts/erc-721/erc721base)                   | Implementation of the [ERC721A](https://www.erc721a.org/) standard with the capability to mint NFTs.                                                                                                                                  |
| [ERC721SignatureMint](/extensions/base-contracts/erc-721/erc721signaturemint) | Adds [signature-based minting](/extensions/erc721signaturemintable) capability to the [ERC721Base](/extensions/base-contracts/erc-721/erc721base) contract.                                                                           |
| [ERC721LazyMint](/extensions/base-contracts/erc-721/erc721lazymint)           | Batch lazy-mint NFT metadata and have other wallets claim them under conditions you define in a `verifyClaim` function.                                                                                                               |
| [ERC721DelayedReveal](/extensions/base-contracts/erc-721/erc721delayedreveal) | Adds [delayed reveal](/extensions/erc721revealable) to the [ERC721LazyMint](/extensions/base-contracts/erc-721/erc721lazymint) contract.                                                                                              |
| [ERC721Drop](/extensions/base-contracts/erc-721/erc721drop)                   | Batch lazy-mint NFTs, and set up [claim conditions](/extensions/erc721claimablewithconditions) to define how your NFTs can be claimed. Also includes [delayed reveal](/extensions/base-contracts/erc-721/erc721delayedreveal) option. |

### ERC1155

| Name                                                                             | Description                                                                                                                                                                                                                              |
| -------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [ERC1155Base](/extensions/base-contracts/erc-1155/erc1155base)                   | Implementation of the [ERC1155](https://eips.ethereum.org/EIPS/eip-1155) standard with the capability to mint and burn NFTs.                                                                                                             |
| [ERC1155SignatureMint](/extensions/base-contracts/erc-1155/erc1155signaturemint) | Adds [signature-based minting](/extensions/erc1155signaturemintable) capability to the [ERC1155Base](/extensions/base-contracts/erc-1155/erc1155base) contract.                                                                          |
| [ERC1155LazyMint](/extensions/base-contracts/erc-1155/erc1155lazymint)           | Batch lazy-mint NFT metadata and have other wallets claim them under conditions you define in a `verifyClaim` function.                                                                                                                  |
| [ERC1155DelayedReveal](/extensions/base-contracts/erc-1155/erc1155delayedreveal) | Adds [delayed reveal](/extensions/erc1155revealable) to the [ERC721LazyMint](/extensions/base-contracts/erc-1155/erc1155lazymint) contract.                                                                                              |
| [ERC1155Drop](/extensions/base-contracts/erc-1155/erc1155drop)                   | Batch lazy-mint NFTs, and set up [claim conditions](/extensions/erc1155claimablewithconditions) to define how your NFTs can be claimed. Also includes [delayed reveal](/extensions/base-contracts/erc-1155/erc1155delayedreveal) option. |

### ERC20

| Name                                                                                | Description                                                                                                                                                                                                                                 |
| ----------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [ERC20Base](/extensions/base-contracts/erc-20/erc20base)                            | Implementation of the [ERC20](https://eips.ethereum.org/EIPS/eip-20) standard with support for the [EIP-2612](https://eips.ethereum.org/EIPS/eip-2612) standard.                                                                            |
| [ERC20Vote](/extensions/base-contracts/erc-20/erc20vote)                            | Add [vote](https://github.com/thirdweb-dev/contracts/blob/main/contracts/openzeppelin-presets/token/ERC20/extensions/ERC20Votes.sol) capability to the [ERC20Base](/extensions/base-contracts/erc-20/erc20base) contract.                   |
| [ERC20SignatureMint](/extensions/base-contracts/erc-20/erc20signaturemint)          | Adds [signature-based minting](/extensions/erc20signaturemintable) capability to the [ERC20Base](/extensions/base-contracts/erc-20/erc20base) contract.                                                                                     |
| [ERC20SignatureMintVote](/extensions/base-contracts/erc-20/erc20signaturemint-vote) | Add [vote](https://github.com/thirdweb-dev/contracts/blob/main/contracts/openzeppelin-presets/token/ERC20/extensions/ERC20Votes.sol) capability to the [ERC20SignatureMint](/extensions/base-contracts/erc-20/erc20signaturemint) contract. |
| [ERC20Drop](/extensions/base-contracts/erc-20/erc20drop)                            | Release your token for a price under the criteria set up in [claim conditions](/extensions/erc1155claimablewithconditions).                                                                                                                 |
| [ERC20DropVote](/extensions/base-contracts/erc-20/erc20drop-vote)                   | Add [vote](https://github.com/thirdweb-dev/contracts/blob/main/contracts/openzeppelin-presets/token/ERC20/extensions/ERC20Votes.sol) capability to the [ERC20Drop](/extensions/base-contracts/erc-20/erc20drop) contract.                   |
