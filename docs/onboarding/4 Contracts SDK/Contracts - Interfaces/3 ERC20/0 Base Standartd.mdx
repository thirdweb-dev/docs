## Unlocked Features

By implementing the `ERC20` standard, you unlock the following features in the SDK and dashboard:

| SDK Feature                                                              | Description                                                                       |
| ------------------------------------------------------------------------ | --------------------------------------------------------------------------------- |
| [Metadata](/sdk/interacting-with-contracts/erc20#get-token-metadata)     | View the metadata (name, symbol, etc.) of the token.                              |
| [View Total Supply](/sdk/interacting-with-contracts/erc20#total-supply)  | View the total supply of the token.                                               |
| [View Balance](/sdk/interacting-with-contracts/erc20#view-token-balance) | View how many tokens a wallet address has.                                        |
| [Transfer](/sdk/interacting-with-contracts/erc20#transfer)               | Transfer a specified amount of this token between wallets.                        |
| [Batch Transfer](/sdk/interacting-with-contracts/erc20#batch-transfer)   | View the total supply of the token.                                               |
| [Allowance](/sdk/interacting-with-contracts/erc20#total-supply)          | Permit other addresses to transfer a specified amount of tokens from your wallet. |

## Implementing It Yourself

This section is meant for advanced users who want to write the functionality from scratch.

```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.0;

import "@thirdweb-dev/contracts/openzeppelin-presets/token/ERC20/ERC20.sol";

contract Contract is ERC20 {
    constructor(
        string memory _name,
        string memory _symbol
    )
        ERC20(
            _name,
            _symbol
        )
    {}
}
```
