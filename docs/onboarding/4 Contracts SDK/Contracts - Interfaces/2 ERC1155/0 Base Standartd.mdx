# ERC1155

ERC1155 is the standard for representing "Edition" NFTs
where each NFT in the collection can have more than 1 quantity; also known as "semi-fungible" tokens.

## Unlocked Features

By implementing the `ERC1155` standard, you unlock the following features in the SDK and dashboard:

| SDK Feature                                                                            | Description                                                             |
| -------------------------------------------------------------------------------------- | ----------------------------------------------------------------------- |
| [Get NFT Balance](/sdk/interacting-with-contracts/erc1155#view-nft-balance)            | View the number of NFTs owned by a wallet address                       |
| [View NFT Metadata](/sdk/interacting-with-contracts/erc1155#get-a-single-nft-metadata) | View the token URI and fetch the metadata from IPFS                     |
| [Transfer](/sdk/interacting-with-contracts/erc1155#transfer-a-single-nft)              | Transfer this NFT from one wallet to another                            |
| [Airdrop](/sdk/interacting-with-contracts/erc1155#airdrop-multiple-nfts)               | Airdrop one or multiple NFTs to the provided a list of wallet addresses |

## Implementing It Yourself

This section is meant for advanced users who want to write the functionality from scratch.

```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.0;

import "@thirdweb-dev/contracts/eip/ERC1155.sol";

contract Contract is ERC1155 {
    constructor(
        string memory _name,
        string memory _symbol
    )
        ERC1155(
            _name,
            _symbol
        )
    {}
}
```
