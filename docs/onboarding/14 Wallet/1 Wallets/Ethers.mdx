Connect to your app with any [ethers.js](https://github.com/ethers-io/ethers.js/) signer.

## Usage

```javascript
import { EthersWallet } from "@thirdweb-dev/wallets";

// can be any ethers.js signer
const signer = Wallet.createRandom();
const wallet = new EthersWallet(signer);

wallet.connect();
```

## Methods

Inherhits all the public methods from the [`AbstractWallet`](interfaces/abstract-wallet) class.
