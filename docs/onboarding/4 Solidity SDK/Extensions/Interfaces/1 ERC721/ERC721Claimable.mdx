# ERC721Claimable

Allow other wallets to **claim** NFTs that you have lazy-minted using (using [ERC721LazyMintable](/solidity/interfaces/erc721lazymintable)).

## Unlocked Features

By implementing `ERC721Claimable`, you unlock the following features in the SDK and dashboard:

| SDK Usage                                                                               | Description                                         |
| --------------------------------------------------------------------------------------- | --------------------------------------------------- |
| [Claim NFT](/sdk/interacting-with-contracts/erc721lazymintable#claim-a-lazy-minted-nft) | Claim (mint) a lazy-minted NFT to a wallet address. |
