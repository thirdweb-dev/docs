# ERC1155Burnable

ERC1155Burnable allows the NFTs in the contract to be burned (transferred to a non-recoverable address).

## Unlocked Features

By implementing the ERC1155Mintable contract, you unlock the following features in the SDK and dashboard:

| SDK Usage                                                                                         | Description                                                                          |
| ------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------ |
| [Burn NFTs](/sdk/interacting-with-contracts/erc1155burnable#burn-a-specified-amount-of-a-nfts-1)  | Burn a specific quantity of NFTs                                                     |
| [Burn Batch of NFTs](/sdk/interacting-with-contracts/erc1155burnable#burn-a-batch-of-nfts)        | Burn a specific quantity of multiple NFTs in one transaction                         |
| [Burn From](/sdk/interacting-with-contracts/erc1155burnable#burn-a-specified-amount-of-a-nfts)    | Burn a specific quantity of NFTs from another wallet (requires permission/allowance) |
| [Burn Batch of NFTs From](/sdk/interacting-with-contracts/erc1155burnable#burn-a-batch-of-nfts-1) | Burn a specific quantity of NFTs                                                     |

    constructor(
        string memory _name,
        string memory _symbol
    )
        ERC1155(
            _name,
            _symbol
        )
    {}

    }

    function burnBatch(
        address account, uint256[] memory ids, uint256[] memory values) external override {
        // Your custom implementation here
    }
}
