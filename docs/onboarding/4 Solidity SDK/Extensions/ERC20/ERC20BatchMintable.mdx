# ERC20BatchMintable

Enable minting multiple NFTs at once in a single transaction by Implementing `ERC20`, `IMintableERC20`, and `Multicall`.

This is an extension which is detectable in the dashboard if the smart contract implements the 
[ERC20](/solidity/extensions/erc20), [Multicall](/solidity/extensions/multicall) and [ERC20Mintable](/solidity/extensions/erc20mintable) extensions.

## Usage

This is an example smart contract that will have ERC20BatchMintable detected on it in the dashboard.

```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.0;

import "@thirdweb-dev/contracts/openzeppelin-presets/token/ERC20/ERC20.sol";
import "@thirdweb-dev/contracts/extension/interface/IMintableERC20.sol";
import "@thirdweb-dev/contracts/extension/Multicall.sol";

contract Contract is ERC20, IMintableERC20, Multicall {
    constructor(
        string memory _name,
        string memory _symbol
    )
        ERC20(
            _name,
            _symbol
        )
    {}

    function mintTo(address to, string calldata uri) external override returns (uint256) {
        // Your custom implementation here
    }

}
```
