## Unlocked Features

By implementing the ERC1155SignatureMint contract, you unlock the following features in the SDK and dashboard:

| SDK Usage                                                                           | Description                                                                             |
| ----------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------- |
| [Signature-Based Minting](/sdk/interacting-with-contracts/erc1155signaturemintable) | Generate signatures that allow other wallets can use to mint NFTs into your collection. |
