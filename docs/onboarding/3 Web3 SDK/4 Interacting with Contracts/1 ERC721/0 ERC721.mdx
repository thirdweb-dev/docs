## View NFT Balance

Get a wallet's NFT balance (number of NFTs in this contract owned by the wallet).

<ThirdwebFeatureCodeSnippet
  featureName={"ERC721"}
  languageFunctionMapping={{
    react: "useNFTBalance",
    javascript: "balanceOf",
  }}
/>

## Get NFT Metadata

Read the URI this NFT points to and fetch that data automatically.

If the metadata is stored on IPFS, this function uses our IPFS gateway (or the one you specify) to read the data.

<ThirdwebFeatureCodeSnippet
  featureName={"ERC721"}
  languageFunctionMapping={{
    react: "useNFT",
    javascript: "get",
  }}
/>

## Transfer NFT

Transfer an NFT from the connected wallet to another wallet.

<ThirdwebFeatureCodeSnippet
  featureName={"ERC721"}
  languageFunctionMapping={{
    react: "useTransferNFT",
    javascript: "transfer",
  }}
  showHr={false}
/>
