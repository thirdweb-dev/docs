# ERC1155 - Signature-Based Minting

Use [signature-based minting](/sdk/advanced-features/on-demand-minting) of NFTs in your smart contract.

You can utilize these features of the SDK on your contract if it implements [SignatureMintERC1155](/solidity/extension-contracts/sigminterc1155).

## Generate a signature

Generate a signature that can be utilized to mint an NFT by another wallet.

The signature specifies any information about the NFT such as the metadata and price.

:::info Admin Operation

The generation of signatures requires the wallet to have the `MINTER` role on the contract by default.

:::
