# SDK’s & Web3 API

## Cross Platform SDK’s

thirdweb provides a broad set of SDK’s to support desktop, mobile and gaming applications.

SDK’s are available in a variety of programming languages including:

- [Javascript/Typescript](https://portal.thirdweb.com/typescript)
- [React](https://portal.thirdweb.com/react)
- [Python](https://portal.thirdweb.com/python)
- [Go](https://portal.thirdweb.com/go)
- [React Native](https://portal.thirdweb.com/react-native)
- [Unity](https://portal.thirdweb.com/unity)

## Web3 API (beta)

thirdweb offers a Web3 API based deployment model in addition to the application SDK’s. Web3API supports multiple configurations for integration into applications:

- Self-hosted web3 API server that exposes a REST API
- Managed service hosted by thirdweb
