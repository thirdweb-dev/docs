# UI components

<div className="heroGrid">
  
  <div>
    <p>
        The React SDK provides a set of components that provide convenient utilities for your application.
    </p>

They make it easy for you to do common tasks such as:

- Connect wallets to your application
- Call functions on your smart contract
- Utilize the unlocked features of your [extensions](/contractkit/extensions)
- Render NFTs with the correct element for the metadata

Below is a list of the components that are available in the SDK.

</div>

| Name                                                        | Description                                                             |
| ----------------------------------------------------------- | ----------------------------------------------------------------------- |
| [Connect Wallet Button](/ui-components/connectwalletbutton) | Allow users to connect their wallets to your website.                   |
| [Web3 Button](/ui-components/web3button)                    | Call functions on your smart contract.                                  |
| [NFT Media Renderer](/ui-components/ipfs-media-renderer)    | Fetch the media of an NFT metadata and render the correct HTML element. |
| [IPFS Media Renderer](/ui-components/ipfs-media-renderer)   | Fetch and render any IPFS URL and render the correct HTML element.      |
