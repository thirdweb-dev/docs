### Frontend Applications

Using our React SDK, you can build full frontend applications integrated with Solana functionality. You can clone our ready-to-go React + Solana app template with the following command:

```bash
npx thirdweb create --template next-typescript-solana-starter
```

To learn more about building frontend applications with the Solana SDK, check the our setup guide:
