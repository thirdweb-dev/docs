# Interacting with Programs

Using our Solana SDK, you can connect to any deployed program and start calling functions, whether you need to read data from the blockchain or make transactions.

Once you [setup the SDK](/solana/setup/frontend), any interaction you make with a program will be made from the conencted wallet automatically.

You can use the following setup guides to make sure that you've properly setup the SDK to use:

## Calling Program Functions

You can call any function on your program including functions to read data or write a transaction to the blockchain by providing:

1. The on-chain address of the program to connect to
2. The name of the function you want to call
3. The parameters to pass to the function
