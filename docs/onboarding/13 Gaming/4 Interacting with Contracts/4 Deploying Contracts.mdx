# Deploy Contracts

Deploy smart contracts directly from the SDK using the
connected wallet with the [`Deployer`](/unity/Deployer) class.

## Prebuilt Contracts

Deploy any [prebuilt contract](/pre-built-contracts) such as the
[NFT Collection](/pre-built-contracts/nft-collection) or
[Token](/pre-built-contracts/token) by providing the required metadata of the contract.

| Prebuilt Contract                                      | Method Name         | Required Metadata                                                                                                                        |
| ------------------------------------------------------ | ------------------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| [`NFTCollection`](/pre-built-contracts/nft-collection) | DeployNFTCollection | [`NFTContractDeployMetadata`](/unity/NFTContractDeployMetadata#struct_thirdweb_1_1_n_f_t_contract_deploy_metadata)                       |
| [`NFTDrop`](/pre-built-contracts/nft-drop)             | DeployNFTDrop       | [`NFTContractDeployMetadata`](/unity/NFTContractDeployMetadata#struct_thirdweb_1_1_n_f_t_contract_deploy_metadata)                       |
| [`SignatureDrop`](/pre-built-contracts/signature-drop) | DeploySignatureDrop | [`NFTContractDeployMetadata`](/unity/NFTContractDeployMetadata#struct_thirdweb_1_1_n_f_t_contract_deploy_metadata)                       |
| [`Multiwrap`](/pre-built-contracts/multiwrap)          | DeployMultiwrap     | [`NFTContractDeployMetadata`](/unity/NFTContractDeployMetadata#struct_thirdweb_1_1_n_f_t_contract_deploy_metadata)                       |
| [`Edition`](/pre-built-contracts/edition)              | DeployEdition       | [`NFTContractDeployMetadata`](/unity/NFTContractDeployMetadata#struct_thirdweb_1_1_n_f_t_contract_deploy_metadata)                       |
| [`Pack`](/pre-built-contracts/pack)                    | DeployPack          | [`NFTContractDeployMetadata`](/unity/NFTContractDeployMetadata#struct_thirdweb_1_1_n_f_t_contract_deploy_metadata)                       |
| [`Token`](/pre-built-contracts/token)                  | DeployToken         | [`TokenContractDeployMetadata`](/unity/TokenContractDeployMetadata#struct_thirdweb_1_1_token_contract_deploy_metadata)                   |
| [`TokenDrop`](/pre-built-contracts/token-drop)         | DeployTokenDrop     | [`TokenContractDeployMetadata`](/unity/TokenContractDeployMetadata#struct_thirdweb_1_1_token_contract_deploy_metadata)                   |
| [`Marketplace`](/pre-built-contracts/marketplace)      | DeployMarketplace   | [`MarketplaceContractDeployMetadata`](/unity/MarketplaceContractDeployMetadata#struct_thirdweb_1_1_marketplace_contract_deploy_metadata) |

## Published Contracts

Deploy a contract that has been [published](/publish) by passing
the author and name of the contract.

```csharp
await sdk.deployer.DeployFromContractUri(
    // Wallet address of the author of the contract
    "<publisher-wallet-address>",
    // Name of the published contract
    "<contract-name>",
    // Each value in the list represents a constructor argument
    new object[] { }
);
```

## From Contract URI

```csharp
await sdk.deployer.DeployFromContractUri(
    // URI of your smart contract
    "<your-contract-URI>",
    // Each value in the list represents a constructor argument
    new object[] { }
);
```
