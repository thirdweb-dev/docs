Hook for retrieving the total supply of NFTs claimed from an NFT Drop contract.

Available to use on contracts that implement [`ERC721Claimable`](/contractkit/interfaces/erc721claimable)
or [`ERC1155Claimable`](/contractkit/interfaces/erc1155claimable).

```jsx
import { useClaimedNFTSupply } from "@thirdweb-dev/react";
```

## Usage

Provide your NFT drop contract as the argument.

```jsx
import { useClaimedNFTSupply, useContract } from "@thirdweb-dev/react";

// Your smart contract address
const contractAddress = "{{contract_address}}";

function App() {
  const { contract } = useContract(contractAddress);
  const { data: claimedTokens, isLoading } = useClaimedNFTSupply(contract);
}
```

## Return Value

The hook's `data` property, once loaded, is a `BigNumber`
representing the total supply of NFTs claimed from the NFT drop contract so far.

```jsx
BigNumber;
```
