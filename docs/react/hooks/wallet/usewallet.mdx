Hook to get the instance of the currently connected wallet.

```jsx
import { useWallet } from "@thirdweb-dev/react";
```

## Usage

```jsx
import { useWallet } from "@thirdweb-dev/react";

function App() {
  const wallet = useWallet();
}
```

## Return Value

<details>
<summary>Return Value</summary>
<div>

Returns an instance of one of the supported [wallets](/wallet), or `undefined` if no wallet is connected.

[Learn more about wallets](/wallet).

```typescript
AbstractBrowserWallet<Record<string, any>, Record<string, any>> | undefined;
```

</div>
</details>
